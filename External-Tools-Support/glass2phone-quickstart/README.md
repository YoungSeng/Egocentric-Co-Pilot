# Glass2Phone Quickstart

Curated phone/glasses bridge content copied from `Huawei/Projects/Glass2Phone/quickstart`.

## Included

- `websocket-test/`: Python WebSocket client/server for echo and audio streaming tests.
- `ktor-sample/`: Kotlin Ktor WebSocket server sample source and Gradle metadata.
- `glasses-app/`: Android source for streaming audio from a glasses-side app.
- `phone-server/`: Android phone-side server and audio-to-text source. Large Vosk model assets are excluded.
- `server-app/`: simple Android server app source.
- `voice-to-text/`: Android speech recognition source and upstream README. Demo GIF and generated project folders are excluded.

## Exclusions

Generated build folders, Gradle caches, IDE metadata, local machine properties, model assets, zip files, launcher raster assets, and test fixtures are intentionally omitted.
