# Kotlin MCP SDK Reference

The local source at `Huawei/Projects/MCP/kotlin_mcp/kotlin-sdk` contains the Kotlin Multiplatform MCP SDK. This repository keeps only a reference note rather than copying the full SDK.

Use the Kotlin SDK when an Android, JVM, Wasm, or iOS component needs to act as an MCP client or server directly. It supports resources, prompts, tools, stdio transport, SSE, and Ktor integration.

Relevant copied bridge material lives under `../glass2phone-quickstart/`, where Kotlin/Android samples can be adapted to call or host MCP endpoints.
