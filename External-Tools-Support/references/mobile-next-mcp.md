# Mobile Next MCP Reference

The local source at `Huawei/Projects/MCP/Mobile/mobile-mcp` and `mobile-mc_org` contains Mobile Next MCP material. This repository does not copy the full Node package or `node_modules` tree.

Mobile Next MCP is useful when an agent needs cross-platform mobile automation through structured accessibility snapshots, coordinates, app launch, device selection, screenshots, and ADB-style interactions.

The copied client examples in `../mcp-client-examples/` use Mobile Next style tool names such as `mobile_use_device`, `mobile_launch_app`, `mobile_type_keys`, and `mobile_list_elements_on_screen`.
