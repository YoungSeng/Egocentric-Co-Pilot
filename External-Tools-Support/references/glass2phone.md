# Glass2Phone Reference

The local source at `Huawei/Projects/Glass2Phone` includes larger experiments for local LLMs on Android, Ollama apps, Vosk models, whisper.cpp, and phone/glasses quickstarts.

This repository copies only the quickstart bridge source needed to understand WebSocket audio streaming, Android phone-side serving, glasses-side streaming, and voice-to-text interaction. Heavy model assets and full upstream projects are excluded.

Use `../glass2phone-quickstart/` as a starting point for connecting a wearable frontend to an MCP-capable backend or mobile automation layer.
