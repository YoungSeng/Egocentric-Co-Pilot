# MCP Client Examples


## Files

- `test_mcp_connection.py`: connects to an SSE MCP endpoint and lists available tools.
- `find_video.py`: example flow that selects an Android device, opens YouTube, searches, scans UI elements, scrolls, and taps a matching result.

Both examples expect an MCP endpoint at `http://localhost:6001/mcp`; edit the transport URL for your server.
